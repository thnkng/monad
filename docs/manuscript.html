Manuscript — Documentation Layout

Docs Template

Manuscript Layout

Purpose: Structured knowledge — documentation canvas. Long-form content with navigation.

Use case: Documentation sites, guides, articles, knowledge bases.

Introduction

The Manuscript layout is designed for structured documentation with sidebar navigation and long-form content.

This layout provides a comfortable reading experience with proper typography rhythm, scroll behavior, and section anchors.

"Technology disappears. Experience remains." — Manuscript embodies this principle through invisible structure.

Structure

The layout uses a three-area grid:

Sidebar

Navigation panel with links, labels, and footer. Collapsible on mobile.

Header

Sticky top bar with title and actions. Stays visible during scroll.

Main

Content area with max-width for readability. Proper spacing and rhythm.

Grid Template

grid-template-areas:
  "sidebar header"
  "sidebar main";
grid-template-columns: 280px 1fr;
grid-template-rows: 64px calc(100vh - 64px);

Features

Typography Rhythm

Headings and paragraphs follow a consistent spacing scale:

  • h2: Top 0, bottom space-5
  • h3: Top space-6, bottom space-4
  • h4: Top space-5, bottom space-3
  • p, ul, ol: Bottom space-4, line-height relaxed

Scroll Behavior

Sections use scroll-margin-top to account for the sticky header when anchor links are clicked.

Code Blocks

Inline code like button.hoverable and block code are styled for readability:

.card {
  display: flex;
  flex-direction: column;
  background: var(--color-surface);
  border-radius: var(--radius-lg);
}

Blockquotes

Callouts and quotes stand out with left border and italic text. Perfect for highlighting key concepts.

Examples

Basic Usage

<body class="manuscript">
  <aside class="sidebar">...</aside>
  <header>...</header>
  <main>
    <section id="intro">...</section>
    <section id="content">...</section>
  </main>
</body>

Navigation Links

Use anchor links that match section IDs:

<a href="#introduction" class="sidebar-link hoverable">
  <i data-lucide="book-open"></i>
  <span>Introduction</span>
</a>

Content Cards

Example Card

Cards in the Manuscript layout automatically get proper spacing. First heading has no top margin, last paragraph has no bottom margin.
