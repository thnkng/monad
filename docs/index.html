Características Principais

Tudo que você precisa para construir interfaces elegantes

Atomic Design

Abordagem sistemática de tokens até templates. Construa UIs consistentes com componentes reutilizáveis.

Paleta Emocional

Cores com significado — graphite-mind, cloud-silent, ember-energy. Nomes poéticos que guiam a intenção.

Grid Responsivo

Layouts fluidos que se adaptam graciosamente. Utilitários mobile-first para todos os tamanhos de tela.

Web Components

Elementos interativos com lógica encapsulada. Modals, dropdowns, tabs — prontos para usar.

Silêncio Visual

Elementos são silenciosos por padrão. Interações requerem opt-in com classe .hoverable.

CDN Ready

Adicione duas linhas de código e comece a construir. Sem configuração complexa necessária.

Filosofia

Clareza antes da forma. Cada elemento existe por uma razão. Silêncio visual. O espaço é tão importante quanto o conteúdo. Precisão emocional. Interfaces evocam confiança, leveza, calma.
